# 📝 To-Do List Web App

## 📌 Project Overview
This is a simple To-Do List web application built using HTML, CSS, and JavaScript.  
It allows users to add, manage, and delete daily tasks efficiently.

This project was developed as part of the **CodeAlpha Web Development Internship**.

---

## 🚀 Features

- ➕ Add new tasks
- ✔ Mark tasks as completed
- ❌ Delete tasks
- 🧠 Simple and clean UI
- ⚡ Real-time task updates
- (Optional) Local storage support for saving tasks

---

## 🛠 Technologies Used

- HTML5 (Structure)
- CSS3 (Styling)
- JavaScript (Functionality & Logic)

## ⚙️ How to Run the Project

### Method 1:
1. Download or clone the repository
2. Open the folder
3. Open `index.html` in any browser

### Method 2 (Recommended):
1. Open project in VS Code
2. Install **Live Server extension**
3. Right click on `index.html`
4. Click **Open with Live Server**

---

## 🧠 How It Works

1. User enters a task in the input field
2. Clicks on the **Add** button
3. Task is added to the list dynamically
4. User can:
   - Mark task as done
   - Delete task from the list
